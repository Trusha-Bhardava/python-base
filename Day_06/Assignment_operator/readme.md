Assignment operation:

   Assignment operator are used to assign values to variable. it perform an operation on the variable while assigning the value.

1. equal operator (=) : assign values to variable
2. addition equal (+=) : add the right operand to left and assgin result to left operand.
3. substraction equal (-=) : sustacts right operand by left operand and assign the result to left operand.
4. multiplication equal (*=) : multiply leftt operand by right operand and assign the result to left operand.
5. division equal (/=) : divide left by right operand and assign the result to left operand.
6. floor equal (//*) : performs floor on left operand and assign the result to left operand.
7. modulus assignment (%=) : calculates modulus of left operand and assign result to the left operand.
