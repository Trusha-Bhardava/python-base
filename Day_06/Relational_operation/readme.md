Realational operators :
 
   Realational operators in python are used for to compare two avlues and knowing relation between them. 
   These operator return Boolean result, which either True or False.


1. equal to (==) : check if two value are equal
2. Not equal to (!=) : check if two value are not equal.
3. Less than (<) : checks if left operand is less than the right operand.
4. greater than (>) : checks if the left operand is greater than right operand.
5. less than or equal to (<=>) : checks if the left operand is less than or equal to the right operand.
6. Greater than or equal to (>=) : checks if left operand is greater than or equal to the right operand.
