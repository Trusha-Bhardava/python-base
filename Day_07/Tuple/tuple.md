Tuple :

   A tuple is data structure similar to list. there are slighty difference.
   Tuple is immutable. so you can not modify after craeting tuple.
   Tuple are typically used for grouping related data.

Common Tuple Operations :

1. Accessing Tuple element :
      
        Tuple are immutable , so you can only access their element.

2. Tuple Packing and Unpacking :

        You can Pack multiple values into a tuple and unpack them into seperate variable.

3. Concatenating Tuples :

        You can concatenate two or more tuples to create a new tuples.

4. Checking for an element :

        you can check if an element exists in a tuple using in keyword.
