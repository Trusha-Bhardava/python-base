Loop control Statement :

    it is used to modify behavior of loops. In python, there are primary loop control statement are "break" an "continue".

  1. break :

         The "break" statement is used to exit loop prematurely. it can be applied to "for" and "while". it is allowing you to terminate loop when particular condition is met.


  2. Continue : 

          the "continue" Statement is used to skip the current iteration of the loop and processes to next one. it can be used in both "for" and "while" loop.
