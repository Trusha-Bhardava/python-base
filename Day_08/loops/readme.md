Loops in python :

  loops are concept which allow you to perform repetitive task. two type of loops available

1. For loop :

     for loop used to iterate over sequence and execute set of statement. The loop continue until all items in sequence have been processed.
  
  syntax:
         for variable in sequence :

2. While loop :

     While loop continue to execute block of code as long as specified condition is true. it is often used when you don't know howmant times loop should run.

   syntax :
          while condition :
